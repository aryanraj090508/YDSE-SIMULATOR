A Python-based simulation of **Young’s Double Slit Experiment (YDSE)** that visualizes interference patterns using mathematical modeling and graphical plotting.

---

## 📌 Description

This project simulates the wave interference pattern formed in Young’s Double Slit Experiment. It helps in understanding concepts like **constructive and destructive interference**, **fringe formation**, and **intensity distribution**.

---

## 🚀 Features

- 📊 Graphical visualization of interference pattern  
- ⚙️ Adjustable parameters (wavelength, slit distance, screen distance, etc.)  
- 📈 Intensity vs position plotting  
- 🧠 Helps in conceptual understanding of wave optics  

---

## 🛠️ Technologies Used

- Python  
- NumPy  
- Matplotlib
