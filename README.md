# Young Double Slit Experiment (YDSE) Simulator

## Overview
This project is a Python-based simulation of the Young Double Slit Experiment (YDSE). It allows users to visualize and study interference patterns produced by monochromatic light passing through two closely spaced slits.

### Features
- Interactive visualization of interference intensity distribution.
- Real-time adjustment of wavelength, slit separation, and screen distance.
- Comparison of interference patterns for two different wavelengths.
- 2D fringe pattern visualization.

## Theory

Intensity distribution:

I = 4I₀ cos²(πdy / λL)

Fringe width:

β = λL / d
